# Home Assistant Add-on: Wallos

Wallos is an open-source personal subscription tracker available directly in Home Assistant.

## How to use

1. Install the add-on.
2. (Optional) Set `timezone` in add-on configuration, for example `Europe/Warsaw`.
3. Start the add-on.
4. Open Wallos from the add-on page using **Open Web UI**.

## Notes

- The add-on uses ingress by default.
- Wallos stores data in its internal database and logo directories.
